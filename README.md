# 🛒 Madhav Ecommerce Sales Dashboard

## 📊 Project Overview
This interactive Power BI dashboard provides a deep-dive analysis into the sales performance of **Madhav Ecommerce**. As a student specializing in **AI & ML**, I built this project to demonstrate the intersection of data engineering, business intelligence, and exploratory data analysis (EDA).

The dashboard tracks key performance indicators (KPIs) to help stakeholders identify growth opportunities and optimize inventory management based on consumer behavior.

## 🚀 Key Insights Extracted
Based on the visual analysis of the dataset:

* **Profitability:** The store generated a total profit of **37K** on a total sales volume of **438K**.
* **Regional Performance:** **Maharashtra** is the top-performing state, significantly outperforming Delhi and Uttar Pradesh in sales amount.
* **Category Dominance:** **Clothing** is the most popular category, representing **63%** of all quantities sold.
* **Seasonal Trends:** Profitability peaks significantly in **October and November**, coinciding with holiday shopping periods, while dipping in the third quarter.
* **Payment Preferences:** **COD (Cash on Delivery)** remains the primary payment method at **44%**, followed by UPI at **21%**.
* **Top Customers:** High-value customers like **Harivansh** and **Madhav** were identified as the biggest contributors to revenue.

## 🛠️ Technical Stack
* **Tool:** Power BI Desktop
* **Data Cleaning:** Power Query (ETL process for data standardization).
* **Calculations:** DAX (Data Analysis Expressions) for custom measures like **Average Order Value (AOV)** and **Total Profit**.
* **Visualizations:** Donut charts, Bar charts, Stacked area charts, and interactive Quarterly/State slicers.
